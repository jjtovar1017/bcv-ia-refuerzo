## 🎯 **Comandos Útiles**

### **Deploy y Mantenimiento**
```bash
# Redeploy
netlify deploy --prod --dir=dist

# Ver logs
netlify logs

# Abrir sitio
netlify open:site

# Ver variables de entorno
netlify env:list

# Configurar nueva variable
netlify env:set VARIABLE_NAME "value"
```

### **Desarrollo Local**
```bash
# Servidor local
npm run dev

# Build local
npm run build

# Preview local
npm run preview
```
